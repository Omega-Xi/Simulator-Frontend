CONTROLLER

AccountController

Generated reference page. Keep this aligned with your ASP.NET Core controller implementation.

POST/api/account/signup

Create a new account and trigger email verification/admin approval.

POST/api/account/signin?useCookie=true

Authenticate. Browser flow sets refresh token in HttpOnly cookie and returns access token.

POST/api/account/refresh?useCookie=true

Rotate refresh token and return a new access token.

POST/api/account/signout?useCookie=true

Revoke current refresh session and clear cookie.

GET/api/account/details

Return profile, cash balance, holdings, and realized P&L.

PUT/api/account/update

Update name and margin preference.

POST/api/account/change-password

Change password and invalidate sessions.

POST/api/account/forgot-password

Send password reset email.

POST/api/account/reset-password

Confirm reset token and set new password.

POST/api/account/resend-verification

Send a new verification email.

POST/api/account/approval-request

Request admin approval email again.

GET/api/account/verify-email

Verify email by userId and token query parameters.

Authentication

Protected endpoints require Authorization: Bearer <access-token>. Browser refresh flow should use credentials: "include".
